# ipv6

## Description
The homomorphic circuit computes if two IPv6s adresses are equal. The answer is 1 (= true) or 0 (= false).  

## Requirements

* The IPv6s should not be given in compressed format.

## User parameters

* The user has to give two IPv6s.
```sh
$ ./run.sh 3ffe:1900:4545:3:200:f8ff:fe21:67cf 3ffe:1900:4545:3:200:f8ff:fe21:67cf
$ ./run.sh 4ffe:1900:4545:3:200:f8ff:fe21:67cf 3ffe:1900:4545:3:200:f8ff:fe21:67cf
```
